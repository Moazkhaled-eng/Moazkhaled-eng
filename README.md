### 🔹 Adaptive Echo Cancellation for Speakerphone Scenario

Implemented an adaptive acoustic echo cancellation system in MATLAB using LMS and NLMS algorithms to estimate and suppress a synthetic room echo path.

**Highlights:**

- Built LMS and NLMS adaptive filtering loops from scratch
- Analyzed step-size stability, filter length, convergence, ERLE, and residual echo performance
- Achieved **26.31 dB echo reduction using LMS**
- Achieved **26.67 dB echo reduction using NLMS**
- NLMS showed faster convergence than LMS

---

### 🔹 Common-Emitter BJT Amplifier Simulation

Designed and simulated a single-stage common-emitter BJT amplifier in LTspice using a 2N3904 NPN transistor.

**Highlights:**

- Voltage-divider biasing
- DC operating point analysis
- Transient analysis
- AC frequency response analysis
- Achieved approximately **79x transient voltage gain**
- Achieved **38.06 dB midband gain**
- Bandwidth up to nearly **16.01 MHz**
